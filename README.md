### 翻译流程
* 提交某章的某页翻译后，提交人可以申请lock这一章
* lock信息会出现在本页（ - locked by xxx）
* 如果连续两周，这个人提交翻译内容少于一页，该lock自动解除
* 每个人在同一时刻，最多lock一章
